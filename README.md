
To build this project, you will need the following hardware components:

| Component | Quantity |
| :--- | :--- |
| Maker Uno | 1 |
| Breadboard (Half-sized) | 1 |
| 1.3” OLED Display (SH1106) | 1 |
| Ultrasonic Ranging Module (HC-SR04) | 1 |
| PIR Motion Sensor (HC-SR501) | 1 |
| Photoresistor | 1 |
| Transistor (2N2222) | 1 |
| 3V Miniature Brush Motor | 1 |
| Resistor | 2 |
| Jumper Wires | Variable |
